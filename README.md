## üöÄ Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/dylansoG/shop-analyser.git
   cd shop-analyser
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

### Vercel Deployment

1. **Install Vercel CLI**
   ```bash
   npm i -g vercel
   ```

2. **Deploy to Vercel**
   ```bash
   vercel --prod
   ```
